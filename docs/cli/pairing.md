# `hanzo-bot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
hanzo-bot pairing list whatsapp
hanzo-bot pairing approve whatsapp <code> --notify
```
