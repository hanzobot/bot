FAQ

Frequently Asked Questions

Everything you need to know about getting started.

How does the $5 free credit work?
Every new account receives $5 in free credit upon signup. This covers cloud compute, AI model usage, and all platform features. No credit card required. The credit is enough to run a bot for several hours or experiment with all 15 personas and 52+ skills.
Can I bring my own API keys?
Yes! You can use your own API keys from OpenAI, Anthropic, Google, and other providers. When using BYO keys, we charge a 1% cloud management fee instead of per-token pricing. This gives you full control over your AI costs while still benefiting from Hanzo's infrastructure.
Which platforms are supported?
Hanzo Bot deploys natively to Discord, Slack, Telegram, WhatsApp, X/Twitter, iMessage, and the web. It also integrates with 608+ business tools including Google Sheets, GitHub, Salesforce, HubSpot, Notion, and many more via the automation engine.
Is Hanzo Bot open source?
Yes. The core bot framework is fully open source on GitHub at github.com/hanzoai/bot. You can self-host, modify, and contribute. The managed cloud service adds hosting, scaling, monitoring, and enterprise features on top of the open source core.
What's the difference between Cloud and Always-On?
Cloud ($1/hr per bot) runs your agents on shared infrastructure with auto-scaling â€” you pay only when active. Always-On ($499/mo) gives you a dedicated Mac instance running 24/7 with full desktop access, native apps, and the ability to log into your own services.
How do I deploy a bot?
Three ways: (1) Use the web dashboard at hanzo.bot to configure and launch visually, (2) Use the CLI with npx hanzo-bot deploy for developer workflows, or (3) Use the Hanzo App desktop client for a native experience. All methods support one-click deployment to Hanzo Cloud.
